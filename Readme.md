# Stasik's Killer

2D шутер на выживание. Игрок отбивается от бесконечных волн "Стасиков" (тараканов), стараясь набрать как можно больше очков опыта. 

## Стек

- C++17, Qt 6.8, QML
- CMake

## Как собрать проект

Для сборки проекта потребуется:
- Компилятор C++ (GCC, MinGW или MSVC)
- CMake (версии 3.16 или выше)
- Установленный фреймворк **Qt 6.8** (с модулем *Qt Quick*)

### Через Qt Creator

1. Откройте Qt Creator.
2. Выберите `File -> Open File or Project...` и укажите путь к `CMakeLists.txt` в корне репозитория.
3. Выберите нужный Kit, например *Desktop Qt 6.8.0 MinGW 64-bit*.
4. Нажмите `Run` (Ctrl+R).
